🚀 So testen Sie den Admin-Bereich:

Option 1: Direkter PHP-Aufruf (empfohlen)

Führen Sie diesen Befehl in Ihrem Terminal aus:

cd /home/work/workspace/Projects/web/smallCMS-1 && php -S localhost:8080 -t public/

Dann öffnen Sie in Ihrem Browser:

http://localhost:8080/admin

Option 2: Test-Skript

Sie können auch das Test-Skript verwenden:

/test_admin.php

Option 3: Manuelle Überprüfung

Führen Sie das Debug-Skript aus:

cd /home/work/workspace/Projects/web/smallCMS-1 && php debug_admin.php

⚠️ Wichtige Hinweise:

  • Entwicklungsmodus: Die Authentifizierung ist derzeit deaktiviert.
  • Datenpersistenz: Änderungen werden derzeit nicht gespeichert (nur Demo).
  • Server-Problem: Falls der Server nicht startet, könnte Port 8080 belegt sein.
  • Alternative Ports: Versuchen Sie php -S localhost:8081 -t public/ oder php -S localhost:8888 -t public/
